
# 🌍 Open-Terraria (Mine-Python-Engine)

![Python Version](https://img.shields.io/badge/python-3.14+-blue.svg)
![Framework](https://img.shields.io/badge/framework-pygame--ce-green.svg)

Модульный и высокооптимизированный 2D-движок песочницы, написанный на **Python** с использованием библиотеки **Pygame-CE** и **Pillow**. Движок поддерживает динамическое масштабирование кастомных текстур, процедурную генерацию мира и полноценную систему строительства.

---

## ✨ Основные особенности движка

* 🗺 **Процедурные пещеры**: Генерация рельефа с холмами, лесами и разветвленной системой подземных пещер и пустот.
* 🛠 **Продвинутый крафт (Клавиша E)**: Интерактивное меню создания предметов с автоматической проверкой ресурсов и поддержкой русского языка.
* 🎨 **Умный текстур-пак**: Движок автоматически обрабатывает рисунки любого разрешения (даже холсты из MS Paint) — сам обрезает пустые поля, сжимает до игрового размера 40x40 пикселей и на лету удаляет белый фон, делая его прозрачным.
* 🖱 **Удобный хотбар**: Переключение между инструментами и блоками как с помощью цифр `1-9`, так и плавным **скроллом колесика мыши**.
* 📦 **Архитектура под модификации**: Изолированные папки `textures` и `Mods` позволяют на лету менять графику и добавлять контент без вмешательства в ядро кода.
* 🚀 **Полная переносимость**: Логика путей оптимизирована для беспроблемной компиляции скрипта в готовый одиночный файл `game.exe`.

---

## 🕹 Управление в игре

* **`A` / `D`** (или стрелочки) — Движение персонажа влево и вправо.
* **`Space` / `W` / `Up`** — Прыжок.
* **`Колесико мыши`** (или кнопки `1-7`) — Быстрое переключение слотов хотбара.
* **`ЛКМ (Левая кнопка мыши)`** — Разрушение блоков активным инструментом (Кирка копает камень/землю, Топор рубит дерево/листву). Сломанные блоки автоматически собираются в инвентарь.
* **`ПКМ (Правая кнопка мыши)`** — Установка выбранного из хотбара строительного блока в мир.
* **`E`** — Открыть / закрыть интерактивное меню крафта.

## 🛠 Инструкция по сборке и запуску

### 1. Установка зависимостей
Для работы исходного кода требуются библиотеки `pygame-ce` (современная замена старого pygame для Python 3.14) и `pillow` для обработки изображений:

```bash
pip install pygame-ce pillow
```

### 2. Структура проекта
Убедись, что твои `.png` рисунки лежат в папке `textures` рядом с исполняемым файлом:
```text
game/
│
├── textures/
│   ├── grass.png
│   ├── dirt.png
│   ├── stone.png
│   ├── wood.png
│   ├── leaves.png
│   ├── wooden_pickaxe.png
│   └── wooden_axe.png
│
├── Mods/
└── game.exe
```

### 3. Запуск
```
2 клик по исполняемому файлу или значку
```
